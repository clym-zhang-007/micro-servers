# function-call-qwen-agent

该目录用于存放基于 Qwen Agent 的函数调用示例代码。

## 文件说明

- `assistant_ticket_bot-1.py`：基础版门票助手，包含 SQL 查询工具
- `assistant_ticket_bot-2.py`：在基础版上增加了图表生成与图片输出能力
- `assistant_ticket_bot-3.py`：进一步优化了图表绘制逻辑，并增强了 Agent 能力
- `requirements.txt`：当前示例所需的 Python 依赖

## 使用说明

运行前请先配置好必要的环境变量，例如 `DASHSCOPE_API_KEY`，并根据 `requirements.txt` 安装相关依赖。
